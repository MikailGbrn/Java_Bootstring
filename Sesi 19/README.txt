==== DEMO HELLO DENGAN SWAGGER DAN JAVADOC ====

Melakukan inisiasi swagger sebagai API Interface untuk menampilkan data CRUD dari API.
komponen yang dibutuhkan untuk menginisiasi swagger diantaranya:

Class
---

- Swagger Config
	disimpan di dalam package config, berguna untuk menjalankan swagger dan mengatur
	konfigurasi dari swagger

- Hello Controller
	disimpan di dalam package controller berguna untuk memanggil data atau membuat perintah
	untuk menampilkan data

Dependencies
---

- MySQL
- HikariCP
- JPA Configuration
- Swagger
- dan Sagger UI
