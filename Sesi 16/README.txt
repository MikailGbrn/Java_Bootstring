===== Spring Boot Mahasiswa ======

Menjalankan hibernate dan menampilkan data dengan html menggunakan thymeleaf

Cara Kerja
---
1. membuat controller mahasiswa di dalam package controller untuk menampilkan html
2. membuat dao.java sebagai repository data yang akan dipanggil untuk dieksekusi
3. membuat file entity mahasiswa yang berisikan data table dari database mahasiswa
4. membuat file html di dalam resource untuk menampilkan data sebagai html
5. melakukan CRUD di dalam html yang telah dibuat
