==== BELAJAR FCM ====

Di dalam project ini dilakukan program untuk menjalankan Firebase Messaging Service
FCM adalah sebuah plugin untuk mengirimkan notifikasi.

Terdapat beberapa class dalam project ini


Class Controller
---
dalam class controller memiliki method untuk mengirimkan notifikasi 

Class Service
---
di dalam kelas service, dilakukan method untuk menginisiasi Firebase
